# **App Name**: iCloud Sentinel

## Core Features:

- Secure Authentication: Secure login with Google authentication and domain restriction (@y99.vn).
- Account Dashboard: Centralized dashboard to view and manage iCloud account credentials.
- Quick Copy: Instantly copy email, password, or phone number to clipboard with toast notifications.
- Password Peek: Temporarily reveal passwords by holding the 'eye' icon.
- Real-time Filtering: Real-time search to filter accounts by username, phone number, or notes.
- AI Password Analysis: Password strength indicator powered by Gemini Flash to analyze password security. This is achieved using a tool that will communicate with the backend, to obtain information to present to the user.
- Data Import/Export: Export account list to CSV, and intelligent import to prevent duplicates.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5), conveying trust and security, yet also playfulness and ease of use.
- Background color: Light-blue (#E3F2FD), creating a clean and modern aesthetic.
- Accent color: A slightly cooler blue, light-cyan (#B2EBF2), highlights key actions and elements.
- Body: 'Inter', sans-serif.
- Headlines: 'Space Grotesk', sans-serif. When combined with Inter it creates a balance between modern and functional.
- Use Lucide React icons throughout the application for a consistent and modern look.
- Employ a clean and intuitive layout, leveraging ShadCN/UI components for a seamless user experience.
- Subtle animations, like row highlighting on hover and toast notifications, enhance user engagement.
